Android 6.0适配权限申请

2019-08-03 code

在Android 6.0 (API 23) 发布之前, 所有的权限都在安装应用的时候显示给用户（在清单文件中声明），用户确认安装则表示全部接受这些权限, 之后无法撤销对这些权限的授权.
Android 6.0开始, 一部分比较危险的权限需要在程序运行时显式弹框，显式请求用户授权。什么时候弹这个框,由应用程序自己决定.
对于其他认为不是很危险的权限，仍然保持原来的做法，在用户安装应用程序时就予以授权。
同时在设置中，对于应用的危险权限，用户可以选择性地进行授权或者关闭。

Permission介绍

官方文档-系统权限
所有需要使用的权限都需要在清单文件中声明：

如果您的应用在其清单中列出正常权限（即，不会对用户隐私或设备操作造成很大风险的权限），系统会自动授予这些权限。如果您的应用在其清单中列出危险权限（即，可能影响用户隐私或设备正常操作的权限），系统会要求用户明确授予这些权限。
Android 发出请求的方式取决于系统版本，而系统版本是应用的目标：

  • 如果设备运行的是 Android 6.0（API 级别 23）或更高版本，并且应用的 targetSdkVersion 是 23 或更高版本，则应用在运行时向用户请求权限。用户可随时调用权限，因此应用在每次运行时均需检查自身是否具备所需的权限。如需了解有关在应用中请求权限的详细信息，请参阅使用系统权限培训指南。
  • 如果设备运行的是 Android 5.1（API 级别 22）或更低版本，并且应用的 targetSdkVersion 是 22 或更低版本，则系统会在用户安装应用时要求用户授予权限。如果将新权限添加到更新的应用版本，系统会在用户更新应用时要求授予该权限。用户一旦安装应用，他们撤销权限的唯一方式是卸载应用。

因此有些app可以通过设置targetSDK为22以下来一次性获取很多权限。

权限等级

系统权限分为几个保护级别。需要了解的两个最重要保护级别是正常权限和危险权限：

  • 正常权限涵盖应用需要访问其沙盒外部数据或资源，但对用户隐私或其他应用操作风险很小的区域。例如，设置时区的权限就是正常权限。如果应用声明其需要正常权限，系统会自动向应用授予该权限。如需当前正常权限的完整列表，请参阅正常权限。
  • 危险权限涵盖应用需要涉及用户隐私信息的数据或资源，或者可能对用户存储的数据或其他应用的操作产生影响的区域。例如，能够读取用户的联系人属于危险权限。如果应用声明其需要危险权限，则用户必须明确向应用授予该权限。

权限组

所有危险的 Android 系统权限都属于权限组。如果设备运行的是 Android 6.0（API 级别 23），并且应用的 targetSdkVersion 是 23 或更高版本，则当用户请求危险权限时系统会发生以下行为：

  • 如果应用请求其清单中列出的危险权限，而应用目前在权限组中没有任何权限，则系统会向用户显示一个对话框，描述应用要访问的权限组，系统只告诉用户应用需要的权限组，而不告知具体权限。例如，如果应用请求 READ_CONTACTS 权限，系统对话框只说明该应用需要访问设备的联系信息。如果用户批准，系统将向应用授予其请求的权限。
  • 如果应用请求其清单中列出的危险权限，而应用在同一权限组中已有另一项危险权限，则系统会立即授予该权限，而无需与用户进行任何交互。例如，如果某应用已经请求并且被授予了 READ_CONTACTS 权限，然后它又请求 WRITE_CONTACTS，系统将立即授予该权限。

即用户同意权限组中的任意一个权限等同于同意整个权限组。

  • ${TIME}：当前系统时间

  • ${DAY_NAME_SHORT}：first 3 letters of the current day name. Example: Mon, Tue, etc.
  • ${DAY_NAME_FULL}：full name of the current day. Example: Monday, Tuesday, etc.
  • ${HOUR}：当前小时
  • ${MINUTE}：当前分钟
  • ${PROJECT_NAME}：project名

听着大蛇团写代码的时候

2019-02-17

最近听了一个SHE全集正序歌单，从她们刚出道每一张组合专辑，到后来各自发展出的单人专辑，以及和别人合作的歌，演唱会的现场，听了好多好多种她们的声音，勾起了很多很多回忆，也知道了很多很多新东西，趁着心情复杂的时候记录一下吧。

《恋人未满》，大概这是SHE最出名歌之一，是她们第一张专辑的的第一首歌，2001年，那时候我刚上小学，对流行音乐毫无印象，脑海里最早的记忆是周杰伦的《霍元甲》。

一直到《Super Star》这张专辑，2003年，才开始出现我有印象的歌。那时候在小学，我和两个关系很好的女同学是死党，于是我们喜欢的idol自然也是三人一组的，SHE成为实质意义上我们的第一个偶像组合，除此之外还有emmmmm猫眼三姐妹好像，飞天小女警？不太记得了。三年级的我还不知道MP3是什么，爸妈的手机还是摩托罗拉，最新的歌来自电视上的音乐节目，SHE的歌则来自死党家里的专辑，《半糖主义》、《IOIO》，《奇幻旅程》中的《茱罗记》、《波斯猫》、《十面埋伏》，《安可！》中的《痛快》、《金钟罩铁布衫》，都依稀记得看过MV，对于还只会看动画片的我来说，已经是很难得的经历了。其中对《茱罗记》的印象最深，过了很久都还会唱副歌部分，记得有一句歌词“所以我们的手臂上，有胎记长得像蝴蝶”，于是我们三个纷纷在自己手上强行找了一个像蝴蝶的胎记…小学每个六一儿童节，每个班都要表演节目，《Super Star》最火的那一年，我们班表演的就是《Super Star》，虽然三个人其中混进去一个男孩子（是我们班唱歌最好的男孩子），那是整个小学最潮的一个节目，超酷！

小学毕业之后，我有了自己的第一个MP3，却没有再刻意去关注SHE的新歌了，但是她们出了专辑之后很火的歌还是会听说，一直到《Play》这张专辑，虽然没有专门听，再看到却发现《听袁惟仁弹吉他》，《五月天》，《老婆》这几首都是熟到不行，也是第一次发现《借口》这首致敬周杰伦的歌，非常有意思。一边听一边翻评论，看大家夸Ella是地表最强女rapper，才想起之前完全没意识到Ella在《听袁惟仁弹吉他》里面唱的rap，她们好棒啊，我想，但是我好像，错过她们最辉煌的时候了呢。

《我的电台》，很有趣的一张专辑，她们每个人有一首专门念的歌，像真的电台节目，放在正式歌曲之前做个引子之类，Ella的《熬夜DJ》，wow，才发现她的低音炮好迷人，加上有一点点调皮的台湾腔。现在听广播剧我也比较喜欢偏中性的攻音，但是小时候，完全没发现啊。

《SHERO》这张专辑出来的时候，也是火到毫不关注的我都知道了。看评论才知道，她们有个“女孩三部曲”，《女孩当自强》、《如果你是女孩》和《像女孩的女人》，其实《大女人主义》和《Shero》，以及更早的《半糖主义》，都有一点儿类似的意思，宣扬独立自主，有自己的坚持，挺有气势的。

中学时期有一个假期回到老家看到有个台在放SHE演的电视剧《真命天女》，我才发现SHE居然还演过电视剧？？？立刻去搜了一下发现还有Ella主演的《蔷薇之恋》和Hebe主演的《斗牛要不要》，虽然当时看了感觉好像有点狗血？？？不过nbcs。看评论思考了好久，团宠是谁，解接是谁，甄哥我还是知道的，加滑是Ella，我在KTV点过好几次《老婆》这首歌，也是这次才知道，Ella作词，Selina作曲，实际上一开始是Ella写给Hebe的Demo，评论里说“这个团三角恋也不是一天两天了”，我哈哈哈哈哈哈哈哈，忍不住想如果我曾经zqsg地粉过她们是什么感觉呢，说不定已经站起CP了，但是我又想不出来我会站谁和谁的CP，她们感觉就像一家人一样，看评论有人站EH的ES的都有，果然加滑是总攻嘛哈哈哈哈，翻到B站上看看视频甜死了简直大口大口喂糖啊，笑完又觉得很不是滋味，好想时光倒流回去好好粉一下这三只，《十七》出来的时候她们已经一起十七年了，是我从小学一年级进化成一只社畜这么久的时间。每个人都变化了很多，长成了不同的样子，可我却整整错过了她们这么多年，错过她们刚刚出来的时候，最辉煌的时候，各自独立发展的时候…

《十七》放完，歌单又回到第一首《恋人未满》，不知道有生之年我还有没有机会看一次她们的演唱会呢。
